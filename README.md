### MY Neovim configuration

This is my personal neovim configuration. I use it for development in Js, TS, Python, Go, and Rust.

### Installations

1. Clone the repository
2. Launch neovim

### Features

- LSP
- Autocompletion
- Harpoon
- Copilot (run :Copilot setup)
- Telescope
- Treesitter
- Neo Tree
- Catppuccin theme
- Lualine
- Lazy Plugins loading
